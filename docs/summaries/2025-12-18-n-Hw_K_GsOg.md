# LangChain 新功能：原生提供者工具徹底改變 AI 代理開發

## 重點摘要
LangChain 的 Christian 介紹了 OpenAI 與 Anthropic 推出的「原生提供者工具」，這些工具與模型深度整合，效能極佳。透過 LangChain 最新的套件，開發者不再需要手動處理 JSON 或串接工具，只需直接將工具綁定至模型，即可快速打造出如操作整個 Cloudflare 平台的強大 AI 代理。

## 故事大綱
- **開場**：Christian 提出一個可能將徹底改變 AI 代理（Agent）建構方式的新概念：由 OpenAI 和 Anthropic 推出的原生提供者工具（Native Provider Tools）。這些工具是模型原生支援的，經過特別訓練與優化，而非通用的函式呼叫。
- **中段**：影片對比了新舊開發方式的差異，舊方法需要手動撰寫 JSON 規格、串接各種轉接器，而新方法只需從 LangChain 的提供者套件中匯入工具，並直接綁定到模型上，過程不僅簡單且型別安全。他以一個實際範例展示其強大之處：建立一個能連接到 Cloudflare 所有 MCP 伺服器的代理，使其能動態探索工具、查詢 DNS 紀錄、分析網站（如找出 Hacker News 的頭條新聞），甚至管理帳戶。
- **結尾**：Christian 總結，這種使用原生、模型優化的工具是開發真實世界多模態代理的未來趨勢。他認為這只是冰山一角，並鼓勵開發者親自嘗試，探索更多可能性。

## 關鍵見解
1.  **原生工具的威力**：AI 模型提供商（如 Anthropic）開始推出模型原生支援的工具集（如 MCP tool set），這些工具經模型特別訓練，執行效率與可靠性遠超通用函式。
2.  **開發流程極度簡化**：開發者不再需要處理複雜的 JSON 規格或手動「黏合」元件。LangChain 的新套件讓工具整合就像匯入函式庫一樣簡單。
3.  **動態工具搜尋**：透過 `tool_search` 這類工具，代理可以在執行時才去尋找最適合的工具，大幅節省了寶貴的上下文視窗空間（Context Window），避免被不必要的工具資訊塞滿。
4.  **強大的實用性**：範例展示了一個代理僅透過幾行程式碼，就能夠操作像 Cloudflare 這樣複雜的平台，從 API 查詢到即時瀏覽網頁，展現了驚人的實用價值。
5.  **未來的開發方向**：直接綁定模型原生支援的工具，是建構生產級別、重工具、多模態代理的必然趨勢，所有開發者都應關注此方向。

## 精彩時刻
- 開場白：「今天我想向各位展示一個可能將徹底改變你建構真實代理方式的東西。」
- 核心概念：「你不再需要黏合各種轉接器。你只需綁定那些模型『真正被訓練來使用』的工具，就能立即獲得真實的代理行為。」
- 實際展示成果：代理成功連上 Hacker News，並回報「GPT 5.2 announcement post」是當時的第一名熱門文章。

# LangChain's Big Update: Native Tools Are Changing How We Build AI Agents

## TL;DR
Christian from LangChain introduces "Native Provider Tools" from OpenAI and Anthropic, which are deeply integrated and optimized for their models. With LangChain's newest packages, developers can skip manual JSON and wiring, directly binding tools to the model to rapidly build powerful agents, such as one capable of operating the entire Cloudflare platform.

## Story Flow
- **Beginning**: Christian introduces a concept that could completely change how AI agents are built: Native Provider Tools from OpenAI and Anthropic. These are not generic function calls but capabilities the models were specifically trained and tuned to use.
- **Middle**: He contrasts the old way (manual JSON schemas, "gluing adapters together") with the new, simplified approach of importing tools from LangChain's provider packages and binding them directly to the model. He demonstrates its power with a practical example: an agent that connects to all of Cloudflare's MCP servers, allowing it to dynamically discover tools, query DNS logs, analyze websites (like finding the top post on Hacker News), and manage an account.
- **End**: Christian concludes that using these native, model-optimized tools is the future for building real-world, multimodal agents. He emphasizes that this is just a glimpse of what's possible and encourages developers to try it themselves.

## Key Insights
1.  **The Power of Native Tools**: AI providers like Anthropic are now shipping toolsets (e.g., the MCP tool set) that their models are natively trained to use, making them far more effective and reliable.
2.  **Development is Radically Simplified**: Developers no longer need to handle complex JSON schemas or manually "glue" components. LangChain's new packages make tool integration as simple as an import.
3.  **Dynamic Tool Search**: Features like `tool_search` allow the agent to find the most appropriate tool at runtime, which saves precious context window space by not having to load unused tool definitions.
4.  **Incredible Practicality**: The demo showcases an agent operating a complex platform like Cloudflare with just a few lines of code, performing tasks from API queries to live web browsing, demonstrating immense real-world value.
5.  **The Future of Development**: Directly binding tools that the model is trained on is presented as the definitive direction for building production-grade, tool-heavy, and multimodal agents.

## Notable Moments
- The opener: "Today I want to show you something that maybe completely changes the way you build real agents."
- The core concept: "You can forget about gluing adapters together. You just bind the tools that the model was literally trained to use and you get real agentic behavior just immediately."
- The live demo result: The agent successfully browses Hacker News and reports back that the "GPT 5.2 announcement post" is currently number one.
